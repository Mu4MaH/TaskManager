Программа служит для управления проектами и задачами в группе разработки

    - XML
    - JSON
    - Files r/w
    - JAX-WS
    - WSDL
    - JDBC
    - MyBatis
    - Hibernate
    - JPA
    - SE Weld-cdi
    - Apache Delta Spike
    - Spring
    - Apache Tomcat
    - Servlets
    - SOAP
    - REST
    - React + Redux front
    - Spring ORM
    - Spring Security 
    - Spring Boot

- Список  реализованных технологий
